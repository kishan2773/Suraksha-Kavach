# Suraksha-Kavach



Suraksha Kavach is an integrated IoT-based Fire and Gas Safety System designed to provide real-time monitoring and automated emergency response for industrial and residential environments. The system leverages a multi-layered architecture to detect hazards, trigger physical countermeasures, and provide instant remote alerts.

Key Technical Features
Intelligent Sensing Layer: Utilizes an ESP32 microcontroller interfaced with high-sensitivity gas and flame sensors to monitor environmental safety.

Automated Response: Features a logic-controlled water sprinkler system that activates immediately upon hazard detection to mitigate fire risk locally.

Real-Time Data Sync: Employs Firebase (Firestore) as a backend to log sensor data and maintain a history of safety events.

Cross-Platform Monitoring: Includes a dedicated Flutter mobile application that allows users to receive instant push notifications and monitor sensor statuses in real-time.
